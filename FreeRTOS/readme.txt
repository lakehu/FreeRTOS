Directories:

+ The ./Source directory contains the FreeRTOS source code, portable for ARM CA53.

+ The ./Demo directory contains a demo application for ARM CA53.

Build and run:

+ Update CrossToolDir in ./Demo/A53_S32G2_GCC/Makefile to path to tool directory NXP GCC 9.2 ARM64

+ To build the demo, run the command "make" in the directory ./Demo/A53_S32G2_GCC/

+ *.elf output will be generated in ./Demo/A53_S32G2_GCC/Output folder.
